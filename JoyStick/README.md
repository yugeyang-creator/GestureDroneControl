# M5CoreS3 无人机控制器

## 项目介绍

这是一个基于M5CoreS3开发板的无人机控制器项目，使用ESP-NOW无线通信协议与无人机进行通信，通过蓝牙接收体感手套控制数据，并利用TensorFlow Lite进行姿态识别，实现对无人机的远程控制。

## 功能特点

- 使用ESP-NOW协议与无人机进行低延迟通信
- 支持蓝牙数据接收与解析
- 集成TensorFlow Lite进行姿态识别
- 实现无人机的起飞、降落、上升、下降等基本操作
- 支持无人机的前后左右移动和转向控制
- 自动与无人机配对功能
- 实时显示无人机状态信息

## 硬件要求

- M5CoreS3开发板
- 支持ESP-NOW协议的无人机（如StampFly）
- 蓝牙设备（用于发送控制数据）

## 软件依赖

- Arduino IDE
- M5CoreS3库
- ESP-NOW库
- MicroTFLite库

## 安装与使用

1. 克隆本仓库到本地
2. 在Arduino IDE中打开JoyStick.ino文件
3. 安装所需的依赖库
4. 将代码上传到M5CoreS3开发板
5. 打开串口监视器，观察配对过程
6. 当控制器与无人机成功配对后，即可通过蓝牙发送控制指令

## 控制指令说明

- **油门指令**：
  - 0x01：起飞
  - 0x02：降落
  - 0x03：上升
  - 0x04：下降

- **转向指令**：
  - 0x05：左转
  - 0x06：右转

- **移动指令**（通过TensorFlow Lite模型推断）：
  - 0x07：向前
  - 0x08：向后
  - 0x09：向左
  - 0x0A：向右

## 工作原理

1. 控制器启动后，会自动搜索附近的无人机并进行配对
2. 配对成功后，控制器通过ESP-NOW协议与无人机建立通信
3. 控制器通过蓝牙接收外部发送的控制数据
4. 接收到的数据经过解析后，通过TensorFlow Lite模型进行姿态识别
5. 根据识别结果和控制指令，生成相应的控制信号发送给无人机
6. 控制器接收并显示无人机返回的状态信息

## 注意事项

- 确保M5CoreS3开发板已正确连接电源
- 确保无人机处于可配对状态
- 配对过程中请保持控制器与无人机的距离不要太远
- 控制无人机时请遵守当地法律法规
