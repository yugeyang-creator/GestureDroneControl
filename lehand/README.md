# LeHand 体感手套控制程序

## 项目介绍

LeHand是一个基于Arduino的体感手套控制程序，由Hiwonder公司开发，于歌洋整合适配项目。该程序通过电位器检测手指弯曲程度，使用MPU6050传感器检测手部姿态，然后通过蓝牙发送控制命令，可用于控制无人机飞行（其他设备如仿生机器人、小车和机械臂等也支持，需要扩展）。

## 功能特点

- 手指弯曲检测：通过电位器实时检测5个手指的弯曲程度
- 手部姿态检测：使用MPU6050传感器检测手部的倾斜角度
- 蓝牙通信：通过蓝牙模块发送控制命令
- 多模式支持：
  ——待扩展
- 按键控制：通过K3按键控制无人机起飞与降落
- LED指示：通过LED灯显示系统状态

## 硬件要求

- Arduino开发板
- MPU6050姿态传感器
- 5个电位器（用于检测手指弯曲）
- 蓝牙模块（HC-05或类似）
- 5个LED指示灯
- 按键（K3）
- 连接线若干

## 软件要求

- Arduino IDE
- 以下库文件：
  - SoftwareSerial
  - Wire
  - MPU6050库
  - LobotServoController库（本项目未使用）

## 硬件连接

- 电位器连接：
  - 拇指：A0
  - 食指：A1
  - 中指：A2
  - 无名指：A3
  - 小指：A6
- 蓝牙模块连接：
  - RX：D11
  - TX：D12
- LED连接：D2-D6
- 按键连接：D7
- MPU6050连接：I2C接口（A4-A5）

## 使用方法

1. 将所有硬件按照上述连接方式连接好
2. 在Arduino IDE中打开`lehand.ino`文件
3. 编译并上传代码到Arduino开发板
4. 打开串口监视器，波特率设置为9600
5. 程序启动后会自动进行电位器校准：
   - LED灯会闪烁，表示正在校准
   - 校准过程中，需要先完全张开手，然后完全握紧手
   - 校准完成后，LED灯将熄灭
6. 按下K3按键可以切换控制模式
7. 通过手部动作和手指弯曲来控制相应的设备

## 数据传输格式

蓝牙数据帧格式：
- 帧头：0x55 0x55
- 数据长度：6
- 命令码：0x42
- 操作指令：2字节
- x倾角：2字节
- y倾角：2字节

## 注意事项

- 首次使用前请确保进行完整的电位器校准
- 蓝牙模块需要配置为主模式
- 确保MPU6050传感器安装牢固，避免在使用过程中晃动
- 如需调整控制灵敏度，可以修改代码中的相关参数

## 扩展功能

程序中预留了`run()`、`run2()`和`run3()`函数，可根据需要扩展其他控制模式。

## 故障排除

- 如果LED灯持续闪烁，可能是校准过程未完成，请确保按照提示完成手掌张开和握紧的动作
- 如果蓝牙连接不稳定，检查蓝牙模块的供电和连接
- 如果姿态检测不准确，可能需要重新校准MPU6050传感器
